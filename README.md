# 🍅 Pomodoro Timer

A clean and simple **Pomodoro Timer** built with vanilla HTML, CSS, and JavaScript. Stay focused and manage your work sessions with ease using the classic 25-minute Pomodoro technique.

---

## 📸 Preview

A centered card UI with a large countdown display and three control buttons — **Start**, **Stop**, and **Reset** — styled with a vibrant blue background.

---

## ✨ Features

- ⏱️ **25-minute countdown** timer following the Pomodoro technique
- ▶️ **Start** — begins the countdown
- ⏹️ **Stop** — pauses the timer at the current time
- 🔄 **Reset** — resets the display back to `25:00`
- Auto-restarts the session when the timer reaches zero
- Minimal, distraction-free UI

## 📖 How to Use

1. Open `index.html` in your browser.
2. Click **Start** to begin the 25-minute countdown.
3. Click **Stop** to pause the timer at any point.
4. Click **Reset** to return the timer back to `25:00`.
5. When the timer hits `00:00`, it automatically resets for the next session.

## 💡 About the Pomodoro Technique

The **Pomodoro Technique** is a time management method developed by Francesco Cirillo. It breaks work into focused 25-minute intervals (called *Pomodoros*), separated by short breaks. This helps improve focus, reduce mental fatigue, and boost productivity.

**Recommended cycle:**
1. 🍅 Work for **25 minutes**
2. ☕ Take a **5-minute** short break
3. Repeat 4 times, then take a **15–30 minute** long break
